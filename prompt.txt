# PHASE 1: Project Structure Discovery

## TASK
Scan the entire project directory and create a complete map of all services, projects, and their locations.

## EXECUTION STEPS

1. **Print current working directory**pwd

2. **List all immediate subdirectories**ls -la

3. **Find all .NET projects**
```bashFind all .csproj files
find . -name "*.csproj" -type fFor each .csproj found, output:

Full path
Project name (from .csproj filename)
Project type (check if contains "Microsoft.AspNetCore.Mvc" → Frontend, else → Backend)


4. **Find all Python services**
```bashFind all requirements.txt or pyproject.toml
find . -name "requirements.txt" -o -name "pyproject.toml" -type fFor each found, output:

Directory path
Service name (directory name)
Main entry point (find main.py or main.py or app.py)


5. **Identify API Gateway**
```bashSearch for YARP config
find . -name "yarp.json" -o -name "appsettings.json" -type f | xargs grep -l "yarp"Output:

Project containing YARP
